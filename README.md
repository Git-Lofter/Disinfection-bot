# Disinfection-bot
Telegram消毒机器人

# 使用说明

1.配置文件内的设置项

```php
define("SLEEPTIME",300);// 消毒时间 单位：S
define("TOKEN","<Bot Token>"); 	//Bot Token
define("CHATID","<群组ID>"); 	//群组ID
```

2.设置定时任务

```shell
php 文件路径/run.php
```

定时任务 间隔时间 为消毒间隔时间 如：每小时执行一次 / 每小时消毒一次。
